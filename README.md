# Проект: Scene-3D

### Обзор

* Интро
* Деплой


**Интро**

В проекте "Scene-3D" реализован эффект 3D сцены с помощью CSS, реализован эффект дождя и возможность открытия на полный
экран с помощью JavaScript

В проекте применяются технологии адаптивной верстки.
